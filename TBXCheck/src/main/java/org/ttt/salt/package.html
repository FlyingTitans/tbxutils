This is the root package of SALT's TBX validation and editing library and applications. It is designed to be usable not only as a stand-alone validation system, but to be linked against to build TBX class graphs in other applications.

Package Specification

Large TBX Files

The TBXFile is a front end interface to the TBX parser and XCS parser system to allow quick parsing and validation of TBX documents. The limitations of the default implementation is that the entire TBX document is loaded into memory during the XML parse and XCS validation phases.

If a large TBX file is to be validated then the following code should be used: 

    File file = new File("SampleTBX.xml");
    TBXFile tbxfile = new TBXFile(file);
    TBXParser tbxparser = tbxfile.getTBXParser();
    TBXDocument tbxdocument = new TBXDocument(false);
    tbxparser.setTBXDocument(tbxdocument);
    tbxparser.addObserver(myObserver);
    tbxfile.parseAndValidate();
