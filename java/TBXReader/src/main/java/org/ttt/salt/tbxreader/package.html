TBXReader is meant to be used in reading TBX and XCS files, and will handle the associated parsing, validation, and conformance checking of those files.

TBXReader may be used with various design patterns for maximum flexibility in integrating the package within your system. It can build objects for use with the observer or iterator design patterns. It can deliver a W3C DOM Document that may be changed with XCS conformance being performed before a change to the document is allowed.

TBXReader is designed to handle small to very large, greater than 4 GiB, TBX filesâ€”through the use of the {@link TBXObserver} and {@link TBXIterator} objects.

Design Patterns

  • Observer: Use the {@link TBXObserver} to implement the Observer design pattern.
  • Iterator: Use the {@link TBXIterator} to implement the Iterator design pattern.
